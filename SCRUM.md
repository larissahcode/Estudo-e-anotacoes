## SCRUM

Tradicional :

- Só permite que o projeto avance quando uma fase está inteiramente completa.
- Demora "N" meses

Ágil :

- Software construído por partes (incremental) e cada parte executa-se uma em um ciclo (iterativo)
- Semanas



### Pilares do SCRUM

- Transparência

- Adaptação

- Inspeção
